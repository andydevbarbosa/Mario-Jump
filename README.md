🍄 Mario Jump

O Mario Jump é um mini game inspirado no universo clássico do Super Mario Bros., desenvolvido utilizando HTML, CSS e JavaScript puro.
O objetivo do jogo é controlar o personagem e pular os obstáculos no momento certo, evitando colisões e tentando alcançar a maior pontuação possível.

Este projeto foi criado com foco em praticar lógica de programação, manipulação do DOM, animações com CSS e interações com JavaScript, simulando a mecânica simples de jogos de plataforma.

🚀 Tecnologias utilizadas

HTML5 – Estrutura da aplicação

CSS3 – Estilização, cenários e animações

JavaScript (Vanilla JS) – Lógica do jogo e detecção de colisões

🎮 Funcionalidades

Movimento de pulo do personagem

Obstáculos animados na tela

Sistema de colisão entre personagem e objeto

Reinício automático ao perder

Animações com CSS para simular movimento contínuo

🧠 Conceitos aplicados

Durante o desenvolvimento foram aplicados conceitos importantes como:

Manipulação do DOM

Uso de eventos do teclado

Animações com CSS

Lógica de colisão

Estruturação de projetos front-end

📚 Objetivo do projeto

Este projeto foi desenvolvido como exercício prático de aprendizado em desenvolvimento web, reforçando conceitos fundamentais de JavaScript e interação entre HTML e CSS na criação de aplicações interativas.
